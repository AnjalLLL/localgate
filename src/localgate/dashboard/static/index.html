Database

The connection is tested before it is saved. On success it is written to localgate.config.json and takes effect after a restart — the running server keeps its current connection until then. Once restarted, every table (keys, usage, history, memory) lives in the new database. The driver must be async: postgresql+asyncpg://, not postgresql://.
